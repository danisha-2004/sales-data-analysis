# Sales Data Analysis Project

## Objective

Analyze sales data to identify trends and generate business insights.

## Project Description

This project focuses on analyzing sales data to understand sales performance, customer behavior, product trends, and regional performance. The dataset was cleaned, analyzed, and visualized using Python libraries to generate meaningful business insights.

## Tools Used

Python

Pandas

NumPy

Matplotlib

Jupyter Notebook

## Dataset

500 sales records containing product, customer, region and sales information.

## Steps Performed

Data Cleaning

Exploratory Data Analysis

Visualization

Business Insights Generation

## Key Insights

• East region generated highest sales revenue

• Home Appliances category contributed highest sales

• April recorded peak sales performance

• High-value customers were identified

## Conclusion

This project helped analyze customer behavior, product performance and sales trends using Python.
